<h1>３　疑問点・質問</h1>

非同期処理を実施するにあたりPromiseオブジェクトの挙動が難しいと感じた。

【イメージ】
Promise((resolve,reject)=>{
  //何から処理1+結果
  resolve(”結果1”)←これは、「何かしら処理1」を終了し、「結果1」をPromiseに返す

  //何かしら処理2+結果2
  reject("結果2")←これは、「何かしら処理2」を終了し、「結果2」をPromiseに返す
})
.then((結果1,結果2)=>{
//Promiseがfullfiledだった時
  結果1に対する処理
  
//Promiseがrejectだった時
  結果2に対する処理
})

Promiseの理解は上記のイメージで、いいんでしょうか？
