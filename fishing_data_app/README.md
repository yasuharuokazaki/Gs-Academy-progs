<h1>２　拘った点</h1>
自分の投稿した情報と、他人が投稿した情報を別の色でピン打ちできるようにしました。これにより、自分の情報と他人の情報が一目で区別できるようになりました。
また、投稿する情報の公開設定をできるようにしました。公開設定しなかった情報は、他人に公開されなくなりました。後から公開設定を変更もできます。
